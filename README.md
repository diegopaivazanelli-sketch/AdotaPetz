# 🐾 AdotaPetz

Projeto desenvolvido com foco em simular uma plataforma de adoção de pets, com o objetivo de conectar pessoas interessadas em adotar animais.

## 🚀 Sobre o projeto

O AdotaPetz foi criado durante aulas práticas de desenvolvimento web, com o objetivo de aplicar na prática conceitos de HTML e CSS.

A proposta é apresentar uma interface simples e intuitiva para visualização de pets disponíveis para adoção.

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  

## 🎯 Funcionalidades

- Estrutura de páginas para apresentação de pets  
- Layout organizado e estilizado  
- Interface amigável para navegação
